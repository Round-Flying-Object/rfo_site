Anim Libri

Anim Libri is a Mac application that integrates AI Q&A into a document reader. It allows you to select any text in the document or conversation and use it at the basis for discussion. It will also automatically generate a couple questions to save typing time.

  • Anim Libri is free for personal use, but does require your own Anthropic API key.
  • The cost for the Anthropic API is minimal, roughly 5 cents per thousand words generated.
  • The application comes in at under a megabyte, and only communicates with the Anthropic API.
  • It currently only handles PDF files - EPUB is probably next, and then maybe some other types or functionality.
